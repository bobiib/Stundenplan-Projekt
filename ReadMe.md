Eine C# Konsolenanwendung, die automatisch Stundenpläne für eine Schule erstellt. Das System arbeitet mit klar strukturierten Stammdaten und nutzt dafür die Angaben zu Lehrpersonen, Klassen, Räumen und Fächern. Die Planerstellung berücksichtigt Verfügbarkeiten und vorhandene Ressourcen. Dadurch entsteht ein Zeitplan, der ohne Überschneidungen funktioniert.

Funktionen
Verwaltung von Lehrpersonen, Klassen, Räumen und Fächern
Einsatz eines heuristischen Verfahrens zur Auswahl geeigneter Termine
Prüfung aller Einträge auf Konflikte und korrekte Raumzuweisungen
Bewertung möglicher Lösungen durch ein Strafpunkt System zur Verringerung von Lücken
Speicherung aller Daten im JSON Format, damit Inhalte dauerhaft erhalten bleiben

Starten
Das Starten erfolgt in der gewünschten C# Oberfläche mit dem Startknopf.
